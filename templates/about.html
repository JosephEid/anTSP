About anTSP

anTSP is a Flask web application which is capable of producing solutions to Traveling Salesman Problem instances using an Ant Colony Opitmisation algorithm.

I, Joseph Christian Eid, developed the anTSP web application as part of my final year Dissertation Project supervised by Dr. Dawn Walker at the University of Sheffield.

The title of the project problem is the Development of a Python-based Teaching Tool: using a Bio-inspired Optimisation Method for solving the Travelling Salesman Problem.
